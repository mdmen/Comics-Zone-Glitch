### Controls

- ↑, ↓, ←, → - Moves
- Space - Pause
- J - Punch/Kick
- K - Jump
- L - Defense
- Esc - Menu

Gamepad is also supported

### Available moves

- **Uppercut** (↑ + J)

### Technical features

- Free to choose 2D Canvas or DOM render
- Load assets only needed for current scene
- Dark and light themes
