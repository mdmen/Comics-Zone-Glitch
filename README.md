## Technical features

- Custom game engine
- Can be rendered using 2D Canvas or DOM
- Loads assets only needed for current scene
- Dark and light themes

## Controls

- ↑, ↓, ←, → - Moves
- Space - Pause
- J - Punch/Kick
- K - Jump
- L - Defense
- Esc - Pause

Gamepad is also supported
